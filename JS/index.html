Atalhos Gerais

ctrl+espaço: chama o autocompletar de um contexto

Vamos ver alguns atalhos legais do VS Code que vão deixar a nossa vida mais alegre, supimpa e feliz

alt+z: Queda automática de linha

alt+setas cima/baixo: Move a seleção para cima/baixo no arquivo. Se nada estiver selecionado, movimenta a linha que o cursor está.

shift+alt+setas cima/baixo: copia a seleção para cima/baixo no arquivo. Se nada estiver selecionado, copia a linha que o cursor está.

ctrl+; : comentário/descomentário

  • Auto Rename Tag
  • Color Highlight
  • Live Server
  • Material Icon Name
  • vscode-pets
  • Power Mode
